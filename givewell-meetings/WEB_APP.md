# Web App - GiveWell Board Meetings

The app is now available as a Progressive Web App (PWA) that works on any device!

## Access the Web App

**URL**: https://parconley.github.io/givewell-board-meetings/

## Install to Your iPhone Home Screen

1. Open Safari on your iPhone
2. Go to https://parconley.github.io/givewell-board-meetings/
3. Tap the Share button (box with arrow)
4. Scroll down and tap "Add to Home Screen"
5. Tap "Add"

The app will now appear on your home screen like a native app!

## Features

✅ **Works anywhere** - No computer needed, no same WiFi requirement
✅ **Install to home screen** - Acts like a native app
✅ **Offline capable** - Data cached for offline use
✅ **Audio playback** - Stream all 43 board meetings
✅ **Progress tracking** - Saves where you left off
✅ **Playback speeds** - 1.0x up to 5.0x
✅ **Document viewing** - Read extracted text inline
✅ **Completely free** - No Apple Developer account needed

## Building & Deploying

The web app automatically deploys to GitHub Pages when you push to `main`.

### Manual Build

```bash
cd givewell-meetings
npx expo export --platform web
```

This creates a `dist` folder with the static website.

### Local Testing

```bash
npx expo start --web
```

Opens at http://localhost:8081

## Technical Stack

- **Framework**: Expo + React Native Web
- **Routing**: expo-router
- **Audio**: expo-av (works on web)
- **Hosting**: GitHub Pages
- **PWA**: Manifest + Service Worker (auto-generated by Expo)

## Notes

- Audio streams from GitHub Release (no file size limits)
- Episode data loads from episodes.json on GitHub
- All progress stored in browser localStorage
- Works on iOS, Android, and desktop browsers
